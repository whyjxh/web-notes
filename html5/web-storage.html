webStorage API 提供了储存机制，通过该机制，浏览器可以安全的存储键值对；
webStorage包括两种机制
1、sessionStorage 该存储在会话期间可用；
2、localStorage 该储存在浏览器关闭之后，下次打开可以继续使用；
webStorage 拥有的方法：
1、setItem;
2、getItem；
3、removeItem；
4、clear；
webStorage和cookie的区别；
1、webStorage拥有更大的储存空间；
2、cookie信息每一次请求会传送到服务器；webStorage不会传送到服务器；
